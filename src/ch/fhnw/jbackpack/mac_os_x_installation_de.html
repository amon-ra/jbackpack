Um rdiff-backup zu installieren, müssen Sie folgende Schritte ausführen:
  1. Installieren Sie Apples Xcode Entwicklungsumgebung,
    entweder von Ihrer OS X DVD oder bevorzugterweise
    die aktuellste Version von Apples Entwicklerseite:
    http://developer.apple.com/tools/xcode/index.html.

  2. Installieren Sie MacPorts von http://www.macports.org.

  3. Öffnen Sie Programme->Dienstprogramme->Terminal und geben Sie die
    folgenden Befehle ein (Sie können sie zeilenweise von hier
    kopieren, in das Terminal einfügen, jeweils mit Enter bestätigen
    und die Ausführung des Befehls abwarten):
    sudo port selfupdate
    sudo port install rdiff-backup
    sudo bash
    echo "setenv PATH $PATH" >> /etc/launchd.conf

  4. Starten sie Ihr System neu.
